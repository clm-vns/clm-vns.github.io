
# CV - Clément Vernaison

Ce dépôt contient le fichier du CV de Clément Vernaison, développeur fullstack axé sur l'accessibilité et l'éthique numérique.
Le fichier est compatible à l'impression sur Firefox afin de générer le fichier PDF.

## Structure du CV

Le CV est structuré en plusieurs sections :
- **Titre et accroche**
- **Expérience professionnelle**
- **Certifications**
- **Formations**
- **Compétences**
- **Centres d'intérêt**

## Fichiers

- `index.html` : Version HTML du CV.
- `CV_Clement_Vernaison.pdf` : Version PDF du CV.

## Comment modifier le CV

1. **Mettre à jour les informations** :
   - Ajouter ou modifier les sections dans le fichier source (HTML).

2**Générer une nouvelle version** :
   - Après modification, exporter le fichier au format PDF pour une version finale.

## Conseils pour le CV

- **Rester concis** : Le CV doit tenir sur une page.
- **Mettre en avant les réalisations** : Utiliser des résultats chiffrés pour illustrer les compétences.
- **Adapter le CV** : Personnaliser le CV en fonction des offres d'emploi.
